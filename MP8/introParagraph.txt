This program solves Sudoku game in the input file using recursion. When the Sudoku is solved, the SolveSudoku function will return 1. When the curretn try is incorrect, the function will return 0, and increment, try another number. If none of the number works, the previous try in incorrect and returns 0. In this way, the function backtracks to the start of the problem and continue trying until it finds the solution.

In CheckSudoku function, Bubblesort checks an array of length n*n. The board is copied n*n times to create n*n many n*n arrays and check them every time they are created. Evey row, column and diagonal is checked.
